# Agent File Handoff

These folders are local intake buckets for agent-produced source material.

- `Content/`: candidate public content or durable source material.
- `Improvement/`: audits, QA notes, implementation suggestions, and bug reports.
- `Archive/`: optional local archive after processing.

The public repository intentionally tracks only the bucket structure. Raw dropped files should not be committed. Summarize accepted outcomes in `.uai/intake-outcome-ledger.uai`, `docs/reports/`, or reviewed long-term memory instead.
