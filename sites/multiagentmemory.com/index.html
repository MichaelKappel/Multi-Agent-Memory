Runtime package

Verification scripts

scripts/ holds local route verification, `.uai` memory audits, packaging, secret scanning, enterprise readiness checks, dogfood runs, and deployment helpers.

Database knowledge

Reviewed knowledge lives in protected MemoryEndpoints.com database records. The anonymous /knowledge route is only an empty authentication shell; every company, workspace, or project page, search result, and external-link record requires a workspace-bound key. Accounts and users can belong to multiple companies, but they do not own knowledge trees.

One canonical report can supply focused pages in multiple taxonomy branches without turning repository documentation into a second knowledge system. First-class lifecycle and authority fields keep current canonical guidance ahead of proposed, historical, superseded, archived, community, or unverified material.

No certification, endorsement, automatic repository writes, or hidden authority is claimed.
